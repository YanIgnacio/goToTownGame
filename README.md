
# Going to Town with my Veggies

This is a small silly game submitted to Jerbob's SILLY JAM, which we took the theme "GO TO TOWN" literally, so we wanted to make a game about taking your veggies to town with your truck. Throughout the adventure, you'll be able to find special gimmicks that may or may not help you get to Town!
